Langkah 6: Buat Training Script
Buat file baru: train_denoiser.py
Struktur training script:
A. Import & Setup

Load model (UNet1D atau SimpleCNN1D)
Setup device (GPU/CPU)
Setup optimizer (Adam, lr=0.001)
Setup loss function (MSE)
Setup learning rate scheduler

B. Training Loop
For each epoch:
    Training phase:
        For each batch:
            1. Load noisy dan clean
            2. Preprocessing (Anscombe + normalize)
            3. Forward pass: prediction = model(noisy)
            4. Calculate loss: MSE(prediction, clean)
            5. Backward pass
            6. Update weights
            7. Log loss
    
    Validation phase:
        For each batch:
            1. Load noisy dan clean
            2. Forward pass (no gradient)
            3. Calculate val loss
            4. Log val loss
    
    Save checkpoint:
        - Save model weights
        - Save optimizer state
        - Save loss history
        
    Early stopping:
        - If val loss tidak turun 10 epoch → stop
C. Logging & Monitoring

Print loss tiap epoch
Save loss history ke CSV
Plot training curves
Save best model (lowest val loss)

### **FASE 5: ITERASI & IMPROVEMENT** 🔧

#### **Langkah 14: Hyperparameter Tuning**

**Kalau hasil kurang memuaskan, coba:**

**A. Architecture:**
- Ganti SimpleCNN → UNet1D (lebih bagus preserve detail)
- Adjust channels (32 → 64 → 128)
- Tambah/kurangi layers

**B. Training:**
- Learning rate: 0.001 → 0.0005 atau 0.0001
- Batch size: 16 → 32 (kalau GPU cukup)
- Epochs: 100 → 150 atau 200

**C. Loss Function:**
- MSE → MAE (lebih robust)
- Tambah Peak Preservation Loss
- Weighted loss (kasih bobot lebih di area puncak)

**D. Data Augmentation:**
- Shift 2theta sedikit (±1°)
- Scale intensity (×0.8 sampai ×1.2)
- Add more noise variations
