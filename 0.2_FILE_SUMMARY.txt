## 📦 DAFTAR FILE YANG DIBUAT

### 1. CORE TRAINING SCRIPTS

#### 4_split_dataset.py
- **Fungsi:** Split dataset menjadi training (80%) dan validation (20%)
- **Input:** data/processed/clean dan data/processed/noisy
- **Output:** data/processed/train/* dan data/processed/val/*
- **Command:**
  ```bash
  python 4_split_dataset.py --clean_dir data/processed/clean --noisy_dir data/processed/noisy
  ```

#### xrd_dataset.py
- **Fungsi:** PyTorch Dataset dan DataLoader untuk XRD data
- **Features:**
  - Anscombe transform untuk variance stabilization
  - Normalization ke range [0, 1]
  - Automatic data resampling ke 8500 points
  - Support for batched data loading
- **Command (test):**
  ```bash
  python xrd_dataset.py
  ```

#### models.py
- **Fungsi:** Definisi arsitektur model
- **Models:**
  - SimpleCNN1D: Fast, moderate performance (~1.5M parameters)
  - UNet1D: Better performance, slower (~8M parameters)
- **Command (test):**
  ```bash
  python models.py
  ```

#### config.py
- **Fungsi:** Configuration management untuk training
- **Settings:**
  - Data paths
  - Model hyperparameters
  - Training parameters
  - Device settings
- **Usage:** Import dan gunakan Config class

#### train_denoiser.py ⭐ MAIN TRAINING SCRIPT
- **Fungsi:** Main script untuk training model
- **Features:**
  - Full training pipeline
  - Automatic checkpointing
  - Early stopping
  - Learning rate scheduling
  - Loss plotting
  - TensorBoard support (optional)
- **Commands:**
  ```bash
  # Test mode (5 epochs, small dataset)
  python train_denoiser.py --test_mode --epochs 5
  
  # Full training
  python train_denoiser.py --model unet --epochs 100 --batch_size 16
  
  # Resume training
  python train_denoiser.py --resume checkpoints/checkpoint_epoch_50.pth
  ```

### 2. EVALUATION SCRIPTS

#### evaluate_model.py
- **Fungsi:** Comprehensive model evaluation
- **Metrics:**
  - SNR (Signal-to-Noise Ratio)
  - PSNR (Peak SNR)
  - MSE (Mean Squared Error)
  - MAE (Mean Absolute Error)
  - SSIM (Structural Similarity Index)
- **Outputs:**
  - detailed_results.csv
  - statistics.csv
  - denoising_examples.png
  - metric_distributions.png
- **Command:**
  ```bash
  python evaluate_model.py \
      --model models/saved/best_model.pth \
      --data_clean data/processed/val/clean \
      --data_noisy data/processed/val/noisy \
      --output_dir evaluation_results
  ```

#### analyze_failures.py
- **Fungsi:** Analyze worst performing samples
- **Features:**
  - Identify failure patterns
  - Plot worst and best cases
  - Generate improvement recommendations
- **Command:**
  ```bash
  python analyze_failures.py \
      --model models/saved/best_model.pth \
      --data_clean data/processed/val/clean \
      --data_noisy data/processed/val/noisy \
      --output_dir failure_analysis
  ```

### 3. INFERENCE SCRIPTS

- **Fungsi:** Unified command-line interface

  - `train` - Train model
  - `evaluate` - Evaluate model
  - `denoise` - Denoise single file

  - `analyze` - Failure analysis

## 🛠️ TROUBLESHOOTING

### CUDA Out of Memory
```bash
# Reduce batch size
python train_denoiser.py --batch_size 8  # or 4
```

### Training Too Slow
- Use GPU! (Google Colab free)
- Or use SimpleCNN: `--model simple_cnn`

### Loss Not Decreasing
- Lower learning rate: `--lr 0.0005`
- Try different model: `--model unet`

### Loss Becomes NaN
- Much lower learning rate: `--lr 0.0001`
- Check data quality

## 📞 SUPPORT

Jika mengalami masalah:
1. Cek error message dengan teliti
2. Verify semua paths benar
3. Test dengan `--test_mode` dulu
4. Check training curves di logs/
